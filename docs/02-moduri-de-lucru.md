# Moduri de lucru

ISYY Business System folosește trei moduri principale de lucru:

1. MOD DISCUȚIE.
2. MOD ANALIZĂ.
3. MOD EXECUȚIE.

Fiecare mod are un scop clar. Separarea lor previne confuzia și lucrul haotic.

## MOD DISCUȚIE

MOD DISCUȚIE se folosește când ideea nu este încă suficient de clară.

În acest mod se pot:

- discuta idei;
- pune întrebări;
- compara variante;
- explora posibilități;
- clarifica obiective;
- identifica probleme generale.

În acest mod nu se:

- implementează;
- creează fișiere fără cerere explicită;
- iau decizii definitive fără analiză;
- pornește dezvoltarea unui proiect.

MOD DISCUȚIE este potrivit pentru începutul unei idei.

## MOD ANALIZĂ

MOD ANALIZĂ se folosește când o idee trebuie evaluată critic.

Structura recomandată este:

- ce este solid;
- ce este slab sau neclar;
- ce lipsește;
- ce recomand;
- ce nu recomand.

În acest mod se pot:

- analiza proiecte;
- evalua riscuri;
- compara alternative;
- verifica fezabilitatea;
- identifica pași următori;
- pregăti o decizie.

În acest mod nu se:

- implementează direct;
- schimbă fișiere fără cerere clară;
- tratează presupunerile ca certitudini;
- promite rezultate financiare.

MOD ANALIZĂ este potrivit înainte de decizii importante.

## MOD EXECUȚIE

MOD EXECUȚIE se folosește când cerința este clară și trebuie executată.

În acest mod se pot:

- crea documentație;
- modifica fișiere;
- scrie cod;
- crea structură;
- implementa specificații;
- rula verificări locale.

În acest mod nu se:

- schimbă direcția fără oprire și confirmare;
- inventează cerințe noi;
- modifică arhitectura stabilită fără aprobare;
- face commit sau push fără cerere explicită;
- șterge structuri importante fără confirmare.

MOD EXECUȚIE este potrivit pentru implementare, documentare și livrare.

## Alegerea modului corect

Se folosește MOD DISCUȚIE când întrebarea principală este: "Ce vrem să facem?"

Se folosește MOD ANALIZĂ când întrebarea principală este: "Merită și cum ar trebui făcut?"

Se folosește MOD EXECUȚIE când întrebarea principală este: "Ce trebuie construit acum?"
