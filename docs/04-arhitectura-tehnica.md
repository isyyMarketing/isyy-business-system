# Arhitectura tehnică

Acest document descrie stack-ul oficial ISYY Ecosystem v1.0 și rolul fiecărei componente.

## Stack oficial

ISYY Ecosystem v1.0 include:

- ChatGPT Plus;
- Codex în VS Code;
- VS Code;
- Docker Desktop;
- GitHub;
- Obsidian;
- Node.js;
- Python;
- Canva;
- Figma;
- Gmail;
- Google Drive;
- Google Calendar;
- Stripe / PayPal după lansare.

## Rolul componentelor

### ChatGPT Plus

ChatGPT Plus este folosit pentru:

- arhitectură;
- analiză de business;
- clarificarea ideilor;
- review tehnic;
- mentorat;
- creare de documentație și specificații.

ChatGPT ajută la gândirea sistemului înainte de implementare.

### Codex în VS Code

Codex în VS Code este programatorul principal.

Este folosit pentru:

- scriere de cod;
- modificarea fișierelor;
- completarea documentației;
- implementarea specificațiilor;
- verificări locale.

Codex lucrează pe baza cerințelor clare.

### VS Code

VS Code este mediul unic de dezvoltare.

Este folosit pentru:

- editarea proiectelor;
- rularea Codex;
- lucrul cu repository-uri;
- organizarea fișierelor;
- verificarea modificărilor.

### Docker Desktop

Docker Desktop este folosit pentru testare și rulare în medii controlate.

Rolul lui este să ajute la:

- rularea aplicațiilor local;
- izolarea dependențelor;
- testarea setup-urilor;
- pregătirea pentru deploy.

### GitHub

GitHub este sistemul oficial de versionare.

Este folosit pentru:

- salvarea codului;
- urmărirea istoricului;
- colaborare;
- backup;
- publicare de proiecte când este cazul.

### Obsidian

Obsidian este folosit pentru documentație strategică și gândire pe termen lung.

Poate conține:

- idei;
- note personale;
- planuri;
- analize;
- decizii strategice.

### Node.js

Node.js este unul dintre limbajele principale pentru proiecte web și automatizări.

Poate fi folosit pentru:

- aplicații web;
- API-uri;
- scripturi;
- backend-uri;
- unelte interne.

### Python

Python este limbaj principal pentru automatizări, procesare de date și AI.

Poate fi folosit pentru:

- scripturi;
- automatizări;
- analiză de date;
- integrare cu servicii externe;
- prototipuri AI.

### Canva

Canva este folosit pentru materiale vizuale rapide.

Poate fi folosit pentru:

- postări social media;
- prezentări;
- materiale de vânzare;
- vizualuri simple pentru proiecte.

### Figma

Figma este folosit pentru design de interfețe și prototipare.

Poate fi folosit pentru:

- wireframe-uri;
- design UI;
- prototipuri;
- componente vizuale;
- clarificarea experienței utilizatorului.

### Gmail

Gmail este canalul principal de comunicare business.

Poate fi conectat cu automatizări pentru:

- emailuri către clienți;
- notificări;
- suport;
- fluxuri de vânzare.

### Google Drive

Google Drive este folosit pentru stocare, organizare și colaborare.

Poate conține:

- documente;
- materiale de business;
- exporturi;
- fișiere pentru clienți;
- resurse de proiect.

### Google Calendar

Google Calendar este folosit pentru organizarea timpului.

Poate susține:

- programări;
- deadline-uri;
- planificare săptămânală;
- întâlniri;
- blocuri de lucru.

### Stripe / PayPal

Stripe și PayPal se adaugă după lansare.

Acestea sunt folosite pentru:

- plăți online;
- abonamente;
- vânzarea de produse digitale;
- încasări pentru servicii.

Nu se integrează înainte ca proiectul să aibă o formă validată și o nevoie clară de monetizare.
