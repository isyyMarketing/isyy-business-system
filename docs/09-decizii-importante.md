# Decizii importante

Acest document conține deciziile oficiale stabilite pentru ISYY Business System v1.0 și ISYY Ecosystem v1.0.

Deciziile de aici sunt baza sistemului inițial.

## Decizii stabilite

### 1. Se folosește ChatGPT Plus

ChatGPT Plus este folosit pentru:

- arhitectură;
- analiză;
- mentorat;
- review tehnic;
- documentație;
- specificații.

### 2. Codex în VS Code este programatorul principal

Codex în VS Code implementează specificațiile primite.

El este folosit pentru:

- cod;
- documentație;
- modificări în repository;
- verificări locale.

### 3. VS Code este mediul unic de dezvoltare

Toate proiectele tehnice sunt dezvoltate în VS Code.

Acest lucru păstrează mediul de lucru simplu și controlat.

### 4. GitHub este sistemul oficial de versionare

GitHub este folosit pentru:

- versionarea codului;
- salvarea istoricului;
- colaborare;
- backup;
- pregătirea lansărilor.

### 5. Docker Desktop este folosit pentru testare și rulare

Docker Desktop este folosit pentru medii locale controlate.

Ajută la testare, rulare și izolarea dependențelor.

### 6. Obsidian este folosit pentru documentație strategică

Obsidian este folosit pentru:

- idei;
- strategie;
- note personale;
- analiză;
- planificare pe termen lung.

### 7. Node.js și Python sunt limbaje principale

Node.js și Python sunt limbajele principale ale ecosistemului.

Node.js este potrivit pentru:

- aplicații web;
- API-uri;
- backend-uri;
- unelte interne.

Python este potrivit pentru:

- automatizări;
- scripturi;
- analiză de date;
- AI;
- prototipuri.

### 8. Canva și Figma sunt folosite pentru design și materiale vizuale

Canva este folosit pentru materiale rapide:

- prezentări;
- postări;
- materiale de vânzare;
- vizualuri simple.

Figma este folosit pentru:

- UI;
- prototipuri;
- wireframe-uri;
- design de aplicații.

### 9. Gmail, Google Drive și Google Calendar sunt conectate pentru business

Aceste servicii susțin organizarea business-ului.

Gmail este folosit pentru comunicare.

Google Drive este folosit pentru fișiere și colaborare.

Google Calendar este folosit pentru programare și organizarea timpului.

### 10. Stripe / PayPal se adaugă după lansare

Stripe și PayPal nu sunt prioritatea inițială.

Ele se adaugă după lansare, când există:

- ofertă clară;
- produs sau serviciu testat;
- nevoie reală de încasare online;
- flux de vânzare suficient de clar.

### 11. ISYY Business System face parte din ISYY Ecosystem v1.0

ISYY Business System este o componentă a ISYY Ecosystem v1.0.

Ecosistemul include uneltele, regulile, rolurile și workflow-urile folosite pentru construirea business-ului online.

## Regulă pentru decizii viitoare

Orice decizie importantă nouă trebuie documentată aici sau într-un document dedicat.

O decizie nedocumentată poate fi uitată, interpretată greșit sau schimbată accidental.
