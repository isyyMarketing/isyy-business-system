## Criterii de validare

Proiectul este validat dacă:

- problema este reală;
- utilizatorul înțelege valoarea;
- există feedback concret;
- există interes pentru plată;
- MVP-ul poate fi îmbunătățit.

## Criterii de lansare

Proiectul poate fi lansat dacă:

- MVP-ul funcționează;
- nu există bug-uri critice;
- documentația minimă există;
- datele sensibile sunt protejate;
- Isac aprobă lansarea.
